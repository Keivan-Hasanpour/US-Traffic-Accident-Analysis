# 🚗 US Traffic Accident Analysis & Visualization

## Overview

This project presents an exploratory data analysis (EDA) and visualization study of traffic accidents in the United States.

The main goal of this project is to analyze accident patterns, identify major factors affecting accident occurrence and severity, and discover geographical and environmental trends using Python data analysis and visualization techniques.

The project uses a large-scale traffic accident dataset and applies data cleaning, statistical analysis, and interactive visualization methods to extract meaningful insights.

---

## Objectives

The main objectives of this project are:

- Analyze accident distribution across different states
- Investigate accident severity patterns
- Explore the relationship between weather conditions and accident severity
- Identify accident hotspots based on geographical locations
- Analyze temporal patterns of accidents by weekday and year
- Study the impact of road infrastructure features on accident severity
- Visualize correlations between environmental and accident-related variables

---

# Dataset

The dataset contains information about traffic accidents in the United States, including:

- Accident location (latitude and longitude)
- Accident severity
- Weather conditions
- Temperature
- Wind speed
- Visibility
- Precipitation
- Distance affected by accident
- Road infrastructure information
- Date and time information
